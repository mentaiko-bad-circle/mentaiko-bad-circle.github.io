スケジュール更新・デプロイの手順書

通常作業

スケジュールを更新してGitHubへ上げる

  1. 作業前に最新を取得する：
    git pull
  2. data/schedule.js をエディタで編集する
  3. ターミナル（Git Bash）で以下を実行：
    git add -A && git commit -m "Update schedule" && git push
  4. pushが完了すれば Netlify 自動デプロイで 1〜2分後に反映

※ Netlify自動デプロイが停止中の場合は下の「即時デプロイ」を使うこと

即時反映

Netlifyに手動デプロイする

  1. ターミナルでプロジェクトフォルダに移動：
    cd D:\Mentaiko
  2. 以下を実行：
    netlify deploy --prod --dir=.
  3. 「Deploy is live!」と表示されれば完了

サイトURL: https://mentaiko-bad-circle.netlify.app

トラブル対応

git push が 403 エラーで失敗したとき

  1. アカウントを切り替える：
    "$LOCALAPPDATA/gh-cli/bin/gh.exe" auth switch --user mentaiko-bad-circle
  2. git認証を再設定：
    "$LOCALAPPDATA/gh-cli/bin/gh.exe" auth setup-git
  3. 再度 push する：
    git push
参考

schedule.js の書き方

項目値の例説明
date"2026-07-04"開催日（YYYY-MM-DD）
time"17:00〜19:00"時間帯
area"京都" / "滋賀"エリア
venue"右京地域体育館"会場名（VENUE_MAPSと一致させる）
fulltrue満員バッジ表示（省略時は募集中）
holidaytrue祝日フラグ（省略可）

全日程の募集を一括停止するには：

const RECRUITMENT_OPEN = false;
